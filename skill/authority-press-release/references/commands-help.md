# Quick Commands

User shorthand. When the user says one of these (or close to it), follow it exactly.

**Start Part 1:** "Start Part 1 for [Project]. Website: [URL]. Agent: [Name] with [Brokerage]..." -> Run the Part 1 research brief.

**Start Part 2:** "Start Part 2" / "Start Part 2 for [Project]" -> Load the saved brief and create the campaign map and headlines first. No releases until approval.

**Approve headlines:** "Approved. Write release [N] using headline option [X]. Keep the angle focused on [angle]." -> Write only the approved releases with the specified headlines and quote direction.

**More headlines:** "Give me more headline options for release [N]. Make them sharper and include [player] and [project]." -> 3-5 new sharper options.

**Different angle:** "Rework the angle. Make it more focused on [buyer concern / local proof / developer credibility / builder reputation / architecture / interiors / lifestyle / affordability / market shift]."

**Restart Part 1:** "Reset Part 1. Start a new Authority Stack Research Brief for [Project]." -> Fresh research, overwrite the saved brief.

**Restart Part 2:** "Reset Part 2. Use the same Part 1 brief but create a new campaign map and new headline options."

**Help choosing:** "Help me choose the best angle based on the buyer avatar, media appeal, and authority players." -> Recommend with reasoning, don't just list.

**Tighten:** "Reduce this to the strongest [1-3] releases. Prioritize the most newsworthy angles and avoid repetition."

**Compliance pass:** "Review for compliance risk." -> Verify the agent reads as a buyer-side local specialist only, never the official project representative.

## Common Fixes

- Too promotional -> rewrite as a real news release, strip marketing copy.
- - Boring headline -> add a stronger buyer question, market shift, or surprising proof point.
  - - Agent sounds affiliated -> reposition as independent buyer-side local specialist commenting on public information.
    - - Too generic -> add more local proof and a clearer buyer avatar connection.
